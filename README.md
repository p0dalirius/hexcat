A tool to show only printable characters of a file

## Features

 - [x] Prints printable chars from a random file or stream

## Usage

```
hexcat <file>
```

## Compilation

Source files are located in `src/` and binary files will go in `bin/`. To compile hexcat, just type :

```
make
```
